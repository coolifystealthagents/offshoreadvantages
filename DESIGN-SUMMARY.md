# Design summary

- Design concept: Orange advantage scorecard
- Layout personality: bold offset scorecards
- Typography: Oswald headings with Source Sans 3 body text
- Color palette: primary #9a3412, secondary #fdba74, accent #0f766e, background #fff7ed, text #1c1917, muted #78716c
- Shape language: Sharp corners
- Visual motif: ranked advantage markers
- Unique layout choices: first sections use Advantage score wall, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
